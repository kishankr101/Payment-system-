# Payment system 

Key Improvements Made:
UI/UX Overhaul:
Glassmorphism: Added backdrop-filter: blur and semi-transparent white backgrounds to cards.
Gradients: Used a professional blue-to-cyan gradient for buttons and a soft grey-blue gradient for the overall page.
Animations: Added hover effects to the country cards (they lift up when you mouse over them).
Typography: Centered titles and used cleaner font-weights.
Search Engine: Added a search bar on the Home page that filters the country list in real-time by Country Name, Region, or Bank Name.
Expanded Data: Added 32 countries with corresponding flags, banks, districts, and actual currency codes.
Updated the rates dictionary to include exchange rates for all added currencies.
Real-World Navigation:
Bottom Nav: A persistent navigation menu (Home, Profile, Logout) is now rendered at the bottom of every page once the user is logged in.
Profile Management: The Profile page now has two states: View Mode (shows a nice card with details) and Edit Mode (allows changing details).
User Flow:
Signup →→ Login →→ Profile Setup →→ Home →→ Payment →→ Verification→→ Success.

Every single page now contains a "Back" or "Return" option.
